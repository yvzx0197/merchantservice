## 两种定价方案：传统模式 vs 零费率模式

MerchantService.com 提供两种完全不同的定价方案，适合不同需求的商家。

### 方案一：Interchange Pass Through（直通交换费）

这是传统的定价模式，但比那种"分层定价"（Tiered Pricing）要透明得多。

简单说就是：他们把银行的成本价（Interchange Fee）原封不动地传递给你，然后只加一点点加价。这个加价幅度据说 15 年没涨过——这在行业里算是比较良心的了。

大多数收单公司都是用"分层定价"，把交易分成"合格"、"中等合格"、"不合格"三档，每档收费不一样。问题是你永远搞不清楚哪笔交易会被分到哪一档，最后账单一来发现被宰了也说不出话。

相比之下，Interchange Pass Through 就是一个费率打天下，清清楚楚。

### 方案二：Cash Discount Program（现金折扣计划）

这个就更狠了——理论上你可以实现零手续费。

怎么做到的？简单说就是把信用卡手续费转嫁给顾客，但方式比较巧妙：

1. 你的标价就是"现金价"
2. 顾客用现金或礼品卡支付 → 享受标准价格
3. 顾客用信用卡支付 → 系统自动加上一个调整费用（通常 3-4%）
4. 你只需要支付固定的月费，无论处理多少交易

这种模式在美国是合法的，因为信用卡网络的规则改了，允许商家对现金和信用卡收取不同价格。

你需要在店里放一个告示牌，告诉顾客"标价是现金价"，刷卡会有额外费用。MerchantService.com 会提供这些标识牌。

**适合谁用？**
- 交易量大但利润薄的商家（比如加油站、便利店）
- 现金客户比例较高的行业
- 希望彻底摆脱手续费困扰的老板

**不适合谁？**
- 高端零售或服务业（顾客可能不爽）
- 电商（线上没法搞"现金折扣"）
- 品牌形象要求高的企业

## 几个需要注意的细节

### 1. 合同条款
虽然他们号称"无隐藏费用"，但还是建议仔细看合同：
- 有没有最低月交易量要求？
- 提前终止合同的费用是多少？
- PCI 合规费怎么算？
- 设备是买断还是租赁？

### 2. Cash Discount Program 的合规性
现金折扣计划在美国大部分州是合法的，但有些州（比如康涅狄格、马萨诸塞）有特殊规定。签合同前确认你所在州的法律。

而且这个计划要求你：
- 在明显位置放置告示牌
- 收据上必须清楚显示现金价和信用卡调整
- 不能对借记卡（Debit Card）加收费用

做不到这些可能会被罚款或者被信用卡网络踢出去。

### 3. 资金到账时间
一般情况下是 T+1 或 T+2（交易后 1-2 天到账）。但如果你是新商户或者被认为是高风险行业，可能会有"滚动储备金"（Rolling Reserve），就是扣留一部分资金作为风险准备金。

这个要提前问清楚，不然突然发现账上少了一大笔钱会很慌。

### 4. 设备所有权
有些收单公司的设备是租的，合同到期要还回去。MerchantService.com 的设备可以买断，但也有租赁选项。买断的话一次性花得多，但长期算下来更便宜。
